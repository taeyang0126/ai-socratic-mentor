# 单主题学习模板（AI 全自动版）

> 适用场景：一个目录只学习一个主题（例如仅学习宏观经济学、Raft、投资学等）。

## 你只需要做两件事

1. 把学习资料放进当前目录（书籍、论文、笔记、链接说明都可以）。
2. 对 AI 说一句：`开始学习`（首次可说：`开始学习 宏观经济学`）。

---

## 一键复制到新目录（四文件）

在模板目录执行：

```bash
TARGET_DIR="/path/to/your/new-study-dir"
mkdir -p "$TARGET_DIR"
cp AGENTS.md README.md SESSION-TEMPLATE.md STUDY-TRACKER-TEMPLATE.md "$TARGET_DIR"/
```

说明：
- 这条命令只复制 4 个核心文件。
- 进入新目录后，直接对 AI 说 `开始学习` 即可。

---

## AI 会自动做什么

每次你说 `开始学习`，AI 必须自动执行：

1. 读取 `progress/study-tracker.md`（若不存在则按模板自动创建）。
2. 读取最近一次会话记录（若当天目录不存在则自动创建；同一天多次学习会创建多个会话文件，按序号递增）。
3. 若是首次学习，先让你确认主题和资料；学习边界与阶段目标由 AI 基于参考学习路线拟定草案并让你确认。
4. 根据追踪器中的「下一次学习入口」继续，不从头开始。
5. 自动确定今天要学的具体内容并开始苏格拉底教学。
6. 结束后自动更新会话记录与学习追踪器。
7. 同步更新本文件中的「学习主题与资料基线（AI 维护）」区块。

## 目录约定

```text
.
├── AGENTS.md
├── README.md
├── SESSION-TEMPLATE.md
├── STUDY-TRACKER-TEMPLATE.md
├── progress/
│   └── study-tracker.md
└── sessions/
    └── YYYY-MM-DD/
        ├── session-001.md
        ├── session-002.md
        └── ...
```

---

## 使用原则

- 不需要手动把模板复制到 `sessions/` 或 `progress/`。
- 不需要你先写学习目标，AI 会基于进度自动生成本次目标。
- 新一天开始学习时，默认延续前一天进度。
- 只有当你明确说「重置/重学」时，AI 才能从头规划。

## 推荐触发语

- 首次：`开始学习 宏观经济学`
- 日常：`开始学习`
- 想加资料后继续：`基于新加的资料继续学习`
