# 单主题学习模板使用说明

> 一套让 AI 成为你的私人导师的模板系统

---

## 快速开始

### 第一步：复制模板到新目录

```bash
# 设置目标目录
TARGET_DIR="/path/to/your/study-topic"

# 创建目录并复制四个核心文件
mkdir -p "$TARGET_DIR"
cp AGENTS.md README.md SESSION-TEMPLATE.md STUDY-TRACKER-TEMPLATE.md "$TARGET_DIR"/

# 进入目录
cd "$TARGET_DIR"
```

### 第二步：放入学习资料

把你的学习资料放进这个目录：
- 书籍/教材 PDF
- 论文/研报
- 笔记文件
- 链接说明文档（写上视频/文档链接）

### 第三步：开始学习

在这个目录下对 AI 说：

```
开始学习 [主题名]
```

例如：
- `开始学习 宏观经济学`
- `开始学习 投资学`
- `开始学习 Raft 共识算法`

## 目录结构

```
your-study-dir/
├── AGENTS.md                    # AI 行为规则（不要修改）
├── README.md                    # 使用说明 + 资料基线（AI 维护）
├── SESSION-TEMPLATE.md          # 会话记录模板（不要修改）
├── STUDY-TRACKER-TEMPLATE.md    # 学习追踪模板（不要修改）
├── progress/
│   └── study-tracker.md         # 学习追踪器（AI 自动创建和维护）
├── sessions/
│   └── 2024-03-10/
│       └── session-notes.md     # 每日会话记录（AI 自动创建）
└── [你的学习资料]
    ├── textbook.pdf
    ├── notes.md
    └── ...
```

## 核心概念

| 概念 | 说明 |
|------|------|
| **学习追踪器** | `progress/study-tracker.md`，记录你的学习进度、里程碑、复习队列 |
| **会话记录** | `sessions/YYYY-MM-DD/session-notes.md`，每天的学习过程记录 |
| **苏格拉底四步** | 探索 → 解释 → 验证 → 跟进，AI 的教学方法 |
| **中断恢复** | 学到一半关掉，下次 AI 会问你要不要继续 |
| **螺旋复习** | 已掌握的主题会定期提醒你复习（7天→14天→30天→90天） |

## 使用流程

### 首次学习

```
你：开始学习 宏观经济学
AI：请确认学习主题和资料清单
你：主题是宏观经济学，资料有曼昆《宏观经济学》和这份课程笔记 [链接]
AI：[给出学习路线草案，让你确认]
你：确认
AI：[开始苏格拉底式教学]
```

### 日常学习

```
你：开始学习
AI：检测到上次学到了 [主题]，已完成 [3/4 步骤]，要继续吗？
你：继续
AI：[从中断处继续教学]
```

### 复习日

```
你：开始学习
AI：检测到 [主题] 已到复习时间，建议先复习，还是学习新内容？
你：先复习
AI：[快速回顾 + 验证问题]
```

## 触发词速查

| 触发词 | 作用 |
|--------|------|
| `开始学习` | 启动学习流程 |
| `开始学习 [主题]` | 首次学习指定主题 |
| `暂停` / `先不学了` | 中断当前会话，下次可继续 |
| `跳过 [主题]` | 跳过已掌握的内容（会有验证题） |
| `这个我会` | 同上 |
| `回退到 [主题]` | 回去重学之前的内容 |
| `前面 [主题] 没懂` | 同上 |
| `重置学习` | 从头开始规划学习路线 |
| `基于新加的资料继续学习` | 加入新资料后继续 |

## 注意事项

### ✅ 推荐做法

- 每次学习前把资料放进目录
- 让 AI 自动管理进度，不要手动改 `progress/` 和 `sessions/`
- 诚实回答验证问题，不懂就说不懂
- 定期查看 `study-tracker.md` 了解自己的学习进度

### ❌ 避免做法

- 不要在同一个目录学习多个主题（会混乱）
- 不要跳过验证问题直接要求讲下一个（AI 会拒绝）
- 不要手动修改进度百分比（AI 会根据里程碑自动计算）

---

## 进度计算规则

```
整体进度 = 已完成里程碑数 / 总里程碑数 × 100%

特殊规则：
- 进行中的里程碑按 50% 计入
- 已跳过的里程碑不计入已完成，但计入总数
```

## 复习间隔规则

| 复习次数 | 间隔 |
|----------|------|
| 首次复习 | 7 天后 |
| 第二次 | 14 天后 |
| 第三次 | 30 天后 |
| 第四次及以后 | 90 天后 |

如果复习未通过，间隔重置为 7 天，重新学习该主题。

## 常见问题

**Q：可以在手机上用吗？**
A：可以，只要你的 AI 客户端支持读取目录中的文件。

**Q：学完一个主题后想学新主题怎么办？**
A：创建新目录，复制四个模板文件，重新开始。

**Q：AI 忘记了之前的进度怎么办？**
A：检查 `progress/study-tracker.md` 是否存在。如果存在，对 AI 说 `开始学习`，它会自动读取。

**Q：可以同时学多个主题吗？**
A：可以，但每个主题需要独立的目录。
